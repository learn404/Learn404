Avant de commencer à coder nous devons setup/mettre en place notre environnement et les outils qui nous permettront de travailler effectivement.

Le premier outil que nous allons devoir installer est vscode, c’est un IDE (***Integrated Development Environment),*** ça va être notre environnement de développement, c’est-à-dire là où l’on va écrire notre code et build nos projets.


❓ **Pourquoi Visual Studio Code ?**

Tout simplement parce que vscode est l’IDE le plus reconnu pour faire du développement web et qu’il est 100% gratuit !
De plus, il contient tout un tas d’extensions qui permettent de coder plus facilement et plus sereinement.



Pour l’installer :

1. Recherchez vscode sur internet ou allez directement à l’url :

[Lien d'installation de VSCODE](https://code.visualstudio.com/)

1. Cliquez sur “Download” en haut à droite puis choisissez quel version vous voulez prendre en fonction de votre OS, si vous être sur Mac, Linux ou Windows. Normalement, vous devrez avoir le téléchargement qui a commencé, et vous avez été redirigés vers la première page de la Doc.
Vous êtes libres de survoler la doc si vous le voulez, ici nous allons passer à la suite et je vais vous expliquer l’essentiel pour continuer le cours.
2. Maintenant allez dans les téléchargements et complétez l’installation.

Maintenant que VsCode est installé et prêt à l’utilisation, on va pouvoir installer quelques extensions qui vont nous faciliter le développement de nos projets :

1. Live Server : lance un serveur local qui se rafraichit à chaque sauvegarde du code.
2. Auto Rename Tag : Renomme automatique le nom de la 2e balise HTML d’un élément lorsque l’on modifie sa première balise.

## Git

Maintenant que l’installation de notre environnement de travail est finie, nous allons installer un deuxième outil qui s’appelle “Git”. Vous avez sûrement déjà entendu ce mot car c’est de là que vient Github, si vous ne connaissez pas, ce n’est pas grave on va regarder ça par la suite.


❓ **Qu’est-ce que Git ?**

Git est un gestionnaire de version décentralisé utilisé principalement pour le suivi des modifications apportées au code source des projets de développement logiciel.

Pour faire simple, il vous permet de garder des traces sur les modifications dans votre code et de gérer des projets en équipe facilement.
Il est décentralisé car chaque développeur possède une copie complète de l'historique du projet sur son propre ordinateur, ce qui rend le système plus résilient et flexible.

❓ **Pourquoi Git ?**

Git est un outil indispensable pour tout développeur, il est utilisé dans la plupart des entreprises et il est très important de savoir s’en servir.


Pour l’installer :

1. Recherchez Git sur internet ou allez à l’url

[Lien d'installation de GIT](https://git-scm.com/)

1. Allez sur “Downloads” et choisissez la version en fonction de votre OS.
2. Terminez l’installation dans vscode.


Maintenant que Git est installé, on va pouvoir commencer à l’utiliser pour gérer nos projets et nos modifications de code.
